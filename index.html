Памятка по информационной безопасности

Информационно-практический проект

Введение

Информационные технологии стали частью повседневной жизни человека. Мы используем интернет для общения, обучения и работы, однако вместе с удобством появляются угрозы: вирусы, мошенничество и кража данных.

Цель проекта — повышение грамотности пользователей в области информационной безопасности и формирование навыков защиты устройств и персональных данных.

Что такое информационная безопасность

  • Конфиденциальность — доступ к информации имеют только разрешённые пользователи.
  • Целостность — данные не изменяются без разрешения.
  • Доступность — информация доступна в нужный момент.

Каждый пользователь сети может стать жертвой кибератаки, поэтому соблюдение правил цифровой безопасности необходимо всем.

Основные угрозы

  • Компьютерные вирусы
  • Сетевые черви
  • Троянские программы
  • Шпионское ПО
  • Программы-вымогатели
  • Рекламное ПО
  • Фишинговые сайты и письма

Большинство заражений происходит из-за невнимательности пользователей: скачивание файлов с неизвестных сайтов, слабые пароли и отсутствие обновлений.

Методы защиты

Технические меры

  • Использование антивируса
  • Включение брандмауэра
  • Регулярные обновления системы
  • Надёжные уникальные пароли
  • Двухфакторная аутентификация
  • Создание резервных копий данных

Организационные меры

  • Не открывать подозрительные письма
  • Скачивать программы только с официальных сайтов
  • Не использовать общественный Wi-Fi для финансовых операций
  • Проверять адрес сайта перед вводом данных
  • Не передавать пароли третьим лицам

Действия при заражении устройства

  1. Отключиться от интернета.
  2. Запустить полное сканирование антивирусом.
  3. Удалить обнаруженные угрозы.
  4. Сменить все пароли.
  5. Обновить систему и программы.
  6. При необходимости восстановить данные из резервной копии.
  7. В крайнем случае — переустановить систему.

Антивирусная защита

  • Kaspersky Anti-Virus
  • Dr.Web
  • Microsoft Defender
  • Malwarebytes
  • Avast Free Antivirus

Ни один антивирус не гарантирует абсолютную безопасность — главную роль играет соблюдение правил цифровой гигиены.

10 правил информационной безопасности

  • Используй сложные пароли.
  • Включай двухфакторную аутентификацию.
  • Устанавливай программы только с официальных сайтов.
  • Регулярно обновляй систему.
  • Не переходи по подозрительным ссылкам.
  • Проверяй адрес сайта.
  • Не используй общественный Wi-Fi для платежей.
  • Создавай резервные копии.
  • Не сообщай пароли и коды.
  • При заражении — отключись от сети и проверь устройство.
